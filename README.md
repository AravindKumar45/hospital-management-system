# # 🏥 Hospital Operation Scheduling System

A responsive frontend web application developed using HTML, CSS, and JavaScript to simplify hospital operation scheduling and emergency appointment management.

## 📌 Project Overview

The Hospital Operation Scheduling System provides an intuitive interface for patients and hospital staff to manage appointments, emergency requests, and scheduling activities efficiently.

This project focuses on improving accessibility and streamlining hospital workflows through a user-friendly web interface.

## ✨ Features

- 🩺 Patient Appointment Scheduling
- 🚑 Emergency Appointment Handling
- 👨‍⚕️ Doctor and Patient Management Pages
- 📅 Date and Time Selection for Appointments
- ⚡ Priority-Based Emergency Requests
- 📱 Responsive User Interface
- 🎨 Clean and Modern Design

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Responsive Web Design

## 🚑 Emergency Handling Module

The emergency handling page allows users to:

- Enter personal details
- Select the type of emergency
- Choose preferred appointment date and time
- Set urgency levels (High, Medium, Low)
- Provide additional information for medical staff

## 🚀 How to Run the Project

1. Clone the repository:

```bash
git clone https://github.com/your-username/hospital-operation-scheduling-system.git
```

2. Open the project folder.

3. Run `index.html` in your browser.
